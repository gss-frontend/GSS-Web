## 🧑🏻‍💻 나만의 개발 회고 저장소, Devoops

Devoops는 개발 중 망각되는 고민들을 자동으로 수집하여 회고를 할 수 있도록 도와주는 웹 기반 AI 회고 서비스 입니다!<br />
다시 코드를 봤을 때 "왜 이렇게 짰었지..? 🤔" 라는 생각이 든적이 있다면, Devoops를 통해 고민들을 아카이빙해 보세요! 👋
